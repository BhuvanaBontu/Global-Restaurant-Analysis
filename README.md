A comprehensive end to end data analysis project using Python for data cleaning and statistical modeling, and Tableau for interactive geographical and performance storytelling of the global restaurant market.
## Project Overview
The Project focuses on a comprehensive analysis of a restaurant dataset to understand market trends, customer preferences, and operational insights across various cities and countries. By combining Python for data processing and statistical analysis with Tableau for interactive visualization, the project provides a data-driven look at the global restaurant industry. 

## Project Objective
* #### Data Cleaning & Exploration
  Identify and handle missing values, duplicates, and data types in the restaurant dataset.
* #### Geographic Analysis
  Map the distribution of restaurants globally and identify top-performing regions.
* #### Customer Sentiment Analysis
  Analyze how ratings correlate with factors like online delivery, table bookings, and cusine types.
* #### Visual Storytelling
  Create a dashboard that effectively communicates key reataurant metrics.

## Dataset Information
The analysis utilizes two primary datasets
* #### data.csv
  Contains 9,551 records and 19 columns, including details like Restaurant ID, Name, City, Cusines, Average Cost for two, and Aggregate Rating.
* #### Country-code.csv
  A mapping file with 15 records used to link "Country Code" to specific country names.
